# 📖 About The Project

**Python Blogs** is a clean and responsive educational website designed for beginners who want to start learning Python Programming.

The website provides a modern interface where users can explore Python tutorials, blogs, and educational content in a simple and visually appealing way.

The main goal of this project is to create a fast, responsive, and beginner-friendly learning platform while practicing modern Frontend Development using HTML and CSS.

# 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/Niloy2004/python-blog.git
```

Go to the project folder

```bash
cd python-blog
```

Run using VS Code Live Server

or simply open

```text
index.html
```
